Version 2.3.0 - Depths of Discovery Major Update

📅 November 15, 2024 📦 Build #2300 💾 2.1 GB

✨ New Features

  • Coral Reef Biome
    Explore the vibrant new coral reef biome featuring over 50 species of marine life, dynamic ecosystems, and hidden treasures waiting to be discovered.
  • Multiplayer Co-op Mode (Beta)
    Team up with up to 4 friends to explore the depths together. Includes synchronized world state, shared discoveries, and cooperative puzzle mechanics.
  • Photo Mode
    Capture stunning underwater moments with our new photo mode. Features include filters, depth of field control, and free camera movement.

🔧 Improvements

  • Enhanced Swimming Mechanics
    Refined swimming controls for more fluid movement. Added momentum-based physics and improved response to ocean currents.
  • Performance Optimization
    Improved rendering performance by 35% in dense coral areas. Reduced memory usage and optimized shader calculations for better frame rates.
  • UI/UX Enhancements
    Redesigned inventory interface with drag-and-drop functionality. Added quick access wheel for frequently used items.

🐛 Bug Fixes

  • Fixed Oxygen Depletion Bug
    Resolved issue where oxygen would deplete rapidly when transitioning between biomes. Oxygen consumption now consistent across all areas.
  • Audio Synchronization
    Fixed underwater audio effects not properly syncing with depth changes. Audio now smoothly transitions based on player position.
  • Save System Reliability
    Addressed rare save corruption issues. Added automatic backup system and improved save validation checks.

⚠️ Breaking Changes

  • Mod API Changes
    Updated mod API to version 2.0. Existing mods will need to be updated to support the new ModLoader.initialize() method.
